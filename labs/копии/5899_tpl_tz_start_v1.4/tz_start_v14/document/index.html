INSTALLATION

You can install this template in two ways:

  • When installing quickstart, you will get the demo data without paid extensions.
  • When installing manually, you need to install your site step by step.
  • If you build your site from scratch, quickstart installation will be the best choice for you.

Quickstart installation

NOTE:

  • You can see how to install our quickstart package on your Cpanel HERE.
  • After the installation, you can have a look at the tutorial of each page to configure the template as you want.

Manual Installation

In this way, you already have Joomla installed and ready to go. To install Joomla manually on a web server, you can follow the instructions in the website: How to Install Joomla?

TZ Start Manual Installation

Follow the steps below to install via Admin Panel (packed template file)

Step 1: Log into the back-end of your site (www.your-site.com/administrator/)

Step 2: Click on: Extensions -> Extension Manager

Step 3: Click on the button "choose the file" in the "Upload & Install Joomla Extension" area

Step 4: Navigate to find the "package_template_zip" file on your computer and click the button "Upload File & Install"

Step 5: Wait while the theme is uploaded and installed.

If the package contains no errors you are done and get a success message like the one beside.

Step 6: Then click on: Extensions -> Template Manager and follow the following instructions like those below.

Basic Things

The header is probably one of the first things you will want to set up. It includes everything above and including the menu, mega menu Configuration, and we will mention creating tags, categories, articles in this section. All of this is done via our Admin Panel. The following sections will cover several different sections of the header that are listed below.

1. Adding a new tag

To create tags which you want to use in each article, you need to install the “TZ Portfolio”.

Step 1: Log into the back-end of your site (www.your-site.com/administrator/)

Step 2: Click on: Extensions -> Extension Manager

Step 3: Click on the button “choose the file” in the “Upload & Install Joomla Extension” area

Step 4: Navigate to find the “tz-portfolio-plus_v_1.1.1.zip” file on your computer and click the button "Upload File & Install"

Step 5: Wait while the theme is uploaded and installed.

If the package contains no errors you are done and get a success message, then you continue to create a new module as normally.

To add a new tag, do the followings:

Step 1: Click on: Components -> TZ Portfolio ->Tags

Step2: Click on the button “New”

Step 3: Complete the fields as required

Name: A proper title for the tag.

Published: Published

Step 4: Click the Save toolbar button to create the new tag

To leave without saving click the Cancel toolbar button

When you go to the Article Manager and select the Section that you created, the tags list will show the tags assigned to that Section

2. Adding a new category

Like creating a new tag, you need to have TZ Portfolio. You can see TZ Portfolio’s installation above.

To add a new tag, do the followings:

Step 1: Click on: Components -> TZ Portfolio ->Categories
Step 2: When the new page loads, click on the “New” button

Step 3: Enter the following fields:


(1)Title: The name of the Category. This is what will show on the pages if needed
(2) Alias: (optional) A unique name for the Category. It is not shown anywhere. If left blank, Joomla will fill it in automatically.
(3)Image: (Optional) displays and Image next to the Category Title on the web pages with the Category Title Showing.
(4) Description: (optional) Describe what the Category is and what it is used for. While this is optional, it is recommended in case other site administrators try to use the Category
Step 4: When all fields have been completed, click the Save button

When you go to the Article Manager and select the Section that you created, the category list will show the Categories assigned to that Section

3. Add a new menu

To add a new menu which can hold menu items do the following:
Step1: Log in to the Joomla! Administrator Back-end.
Step2: Click Menus > Menu Manager on the toolbar menu. The Menu Manager screen opens.

Step3: Click the New toolbar button. The New/Edit Menu screen opens.

Step 4: Complete the fields as required:

Title: A proper title for the menu. This is used to identify the menu the Menu Manager in the backend.

Menu type: This should be a unique identification name used by Joomla! to identify this menu. Spaces are not allowed but you may use the '-' character such as news-menu.

Description: A description of the menu, useful for your own reference.

Step 5: Click the Save toolbar button to create the new menu.

To leave without saving click the Cancel toolbar button

You can now add new menu items to the newly created menu.
